StringLess Harp

Done by
  • A.Ramya Sree
  • K.Naga Sowmya
  • Gaurav
Abstract

Our aim is to make a light sensitive string less harp. Our basic idea involved includes a transparent photo diode sensing the light falling on it. Once the light is blocked the ir gives change in output which is coded accordingly using arduino and converted into sound signals and given to speakers.
Technically, the project is an electronic musical instrument consisting of several light beams to be blocked, in analogy with the plucking of the strings of a harp, in order to produce sounds. It is an autonomous project which works using a microcontroller – ARDUINO UNO and based on simple arduino coding.

Components required
  • Arduino Uno board
  • Bread board
  • Light source
  • Speaker
  • Transparent photo diodes
  • 10ohm and 100k resistors
  • Connecting wires
Connections and procedure

The transparent photodiode is connected to the analog pin a0 which is the input pin. Here photo diode detects the light falling on it and its frequency and then the output processed is given to the analog pin. In this setup an led is used as an indicator for the light detection in the photodiode. The output from the arduino is given as input to the speaker. The coding is done such that when if light falling on it is blocked then the speaker should play the tone assigned. In the caser there is no light falling on the photodiodes the note assigned to the corresponding analog pin is played.

The rest of the photo diodes are also connected similarly to analog pins A1, A2, A3.

In this project we have selected the notes of the violin to be played, assigning each note to each of the four analog pins. The notes are NOTE_G3, NOTE_D4, NOTE_A4, and NOTE_E5. Different tones can be selected from arduino music library. These notes correspond to the notes used to tune the violin strings. The table gives their frequencies of the violin notes.

.

The speaker in connected as shown in the diagram using a 100ohm resistor. The digital pin8 of the arduino i.e., the output pin of the board is connected to the speaker.

Coding

#define NOTE_G3 196
#define NOTE_D4 294
#define NOTE_A4 440
#define NOTE_E5 659
int notes[] = {NOTE_G3,NOTE_D4,NOTE_A4,NOTE_E5};
void setup()
{ pinMode(A0,INPUT);
pinMode(A1,INPUT);
pinMode(A2,INPUT);
pinMode(A3,INPUT);

pinMode(9,OUTPUT);

Serial.begin(9600);
}

void loop()
{
for (int thisSensor = 0; thisSensor < 4; thisSensor++) {
// get a sensor reading:
int sensorReading = analogRead(thisSensor);
Serial.println(sensorReading);
delay(1000);
if(sensorReading <650)
tone(9, notes[thisSensor], 1000);
if (sensorReading == LOW)
{
// play the note corresponding to this sensor:
tone(9, notes[thisSensor-2], 100);//THIS WILL PLAY THE RESPECTIVE NOTE //FOR 100ms*/
}
}
Conclusion

Over the period of time we faced challenges due to insufficient intensity light that is used to focus on the photodiodes. We found that use of laser light will be a better option. A line laser can be used else a point laser can be diffracted accordingly on all the photodiodes. Thus a simple light harp is done using arduino. By adding some extensions other musical instruments can also be made.
